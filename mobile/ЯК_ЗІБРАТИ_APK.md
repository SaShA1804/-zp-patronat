# Як отримати APK для телефону

APK неможливо зібрати на Windows напряму (потрібні Android SDK/NDK + Linux).
Тому збірка йде **безкоштовно у хмарі GitHub Actions** — від вас лише кілька кроків.

## Крок 1. Створити репозиторій на GitHub
1. Зайдіть на https://github.com → **New repository**.
2. Назва будь-яка (напр. `zp-patronat`), тип **Private** — можна.
3. Створіть без README.

## Крок 2. Завантажити проєкт
У папці `приложения  по зп` (де лежать `mobile/` і `.github/`) виконайте в терміналі:

```bash
git init
git add mobile .github
git commit -m "Мобільний застосунок ЗП патронатного вихователя"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГІН/zp-patronat.git
git push -u origin main
```

> Важливо додавати саме `mobile` і `.github` (щоб не залити зайві великі файли з робочого столу).

## Крок 3. Дочекатися збірки
1. Відкрийте репозиторій на GitHub → вкладка **Actions**.
2. Побачите запущений процес **Build Android APK** (стартує сам після `push`;
   також можна запустити вручну кнопкою **Run workflow**).
3. Збірка триває ~15–25 хв (перший раз довше — завантажується Android SDK).

## Крок 4. Завантажити APK
1. Коли з’явиться зелена галочка — відкрийте цей запуск.
2. Внизу, у розділі **Artifacts**, буде `zp-patronat-apk` — завантажте (це .zip з .apk усередині).
3. Розпакуйте, перекиньте `.apk` на телефон.

## Крок 5. Встановити на телефон
1. На телефоні відкрийте `.apk`.
2. Дозвольте «Встановлення з невідомих джерел» (Android попросить сам).
3. Готово — застосунок з’явиться в меню.

---

### Що всередині
- `main.py` — застосунок на Kivy (логіка розрахунку 1:1 з десктопною версією:
  дві категорії ПМ до 6 / 6–18 років, дата народження дитини, надбавка +10% за
  кожну дитину, пільговий 7-денний термін, копіювання результату).
- `buildozer.spec` — параметри збірки APK.
- `../.github/workflows/build-apk.yml` — хмарна збірка.

### Іконка (не обов'язково)
Покладіть файл `icon.png` (512×512) у папку `mobile/` і в `buildozer.spec`
розкоментуйте рядок `icon.filename = %(source.dir)s/icon.png`.
